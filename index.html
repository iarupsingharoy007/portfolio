Linux Server Health Check Automation

A Python tool that connects to a fleet of Linux servers over SSH, runs CPU, memory, disk, process, and service checks, and renders everything into a single-file interactive HTML dashboard.

PythonParamiko / SSHAutomationJinja2
Key highlights
  • Concurrent SSH checks across the whole fleet via ThreadPoolExecutor
  • CPU, memory, swap, disk, mount, process & service validation per server
  • Per-check error isolation — one failing server never blocks the rest
  • Single-file HTML dashboard, no server or database required
